# CyberLens
Lightweight SOC Dashboard for Real-Time Security Monitoring

CyberLens is a lightweight Security Operations Center (SOC) platform designed to collect, centralize, analyze, and visualize security events in real time. The platform aggregates logs from multiple sources, detects suspicious activities and cyberattacks, and provides an interactive dashboard for security monitoring.

-Features:
Real-time log ingestion.
Centralized security event management.
Interactive dashboard and charts
Attack detection and severity classification
Log filtering and search
Real-time dashboard refresh
